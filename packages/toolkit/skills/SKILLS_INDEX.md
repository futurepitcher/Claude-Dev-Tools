# Skills Index

Skills are composable capabilities that can be invoked via slash commands or combined into workflows.

| Skill | Command | Purpose |
|-------|---------|---------|
| **Architecture Planning** | `/plan` | Research-first planning with 3+ approaches, constraint checking, approval gates |
| **TDD Workflow** | `/tdd` | RED-GREEN-REFACTOR cycle with optional phase gate enforcement |
| **Deslop** | `/deslop [path]` | Remove AI-generated cruft — over-engineered abstractions, obvious comments, dead code |
| **Session Management** | `/session` | Save/restore context, create handoff documents, track session metrics |
| **Learner** | `/learner` | Auto-extract non-obvious project-specific insights from debugging sessions |
